# jangbu-jongso

종합소득세·법인세 신고 전 준비 서류 체크리스트.

## 신고 주체별 분기

### 첫 질문 — 신고 주체

```
어떤 신고를 준비하시나요?

[1] 개인사업자 (일반·간이·면세)            → 종소세 5월
[2] 프리랜서 (3.3% 원천징수)                → 종소세 5월
[3] 1인 법인 대표                            → 법인세 3월 + 대표자 종소세 5월
[4] 겸업 (근로소득 + 사업소득)              → 종소세 합산 5월
[5] 성실신고확인대상 개인사업자             → 종소세 6월 (1개월 연장)

번호 입력.
```

### 성실신고확인대상 기준 (2025년 귀속, 업종별 매출액)

- **도·소매업**: 직전 과세기간 수입금액 **15억 원 이상**
- **제조·음식·숙박·건설·운수업**: **7.5억 원 이상**
- **서비스업·부동산임대업**: **5억 원 이상**

본인이 대상인지 모호하면 세무사 확인 필수. 기준 초과인데 일반 신고하면 **가산세 5%** 부과.

### 두 번째 질문 — 귀속 연도

```
어느 연도 귀속분을 신고하시나요?
기본: 2025년 귀속 (2026년 5월 신고)
```

## 유형별 체크리스트

### [1] 개인사업자 — 종합소득세 신고 서류

**A. 소득 자료 (매출)**
- [ ] 매출 세금계산서 (부가세 신고분) — **홈택스 자동 조회**
- [ ] 카드 매출 내역 — 카드사 제공, 홈택스 집계
- [ ] 현금영수증 매출 — 홈택스 자동
- [ ] 직접 계산서 발행분 — 홈택스 자동
- [ ] 수출실적명세서 (수출업자) — 관세청 / 홈택스
- [ ] 간편장부 (간이과세자) — **korean-jangbu-for 자동 생성 가능** ✅

**B. 지출 자료 (필요경비)**
- [ ] 매입 세금계산서 — **홈택스 자동**
- [ ] 매입 계산서 (면세)
- [ ] 지출증빙용 현금영수증 — 홈택스 자동
- [ ] 신용카드 매출전표 — **korean-jangbu-for 자동 분류** ✅
- [ ] 일반 영수증 (3만원 이하) — **OCR 자동 추출** ✅
- [ ] 인건비 지급명세서 (원천징수 이행상황신고서)
- [ ] 퇴직금 지급명세서
- [ ] 기부금 영수증 (지정·법정)

**C. 소득공제·세액공제**
- [ ] 국민연금·건강보험 납부 내역 — 홈택스 자동
- [ ] 신용카드 사용액 (근로소득 있을 때) — 홈택스 자동
- [ ] 의료비·교육비·기부금 — 홈택스 자동
- [ ] 연금저축·IRP — 금융기관 확인서
- [ ] 주택청약종합저축 — 은행 확인서
- [ ] 자녀세액공제 (만 7세 이상 20세 이하)

**D. 추가 자료**
- [ ] 감가상각자산 명세서 — 취득 · 내용연수 · 잔존가액
- [ ] 임차료 지급 내역
- [ ] 차량운행일지 (업무용승용차)
- [ ] 4대 보험료 납부 내역 (종업원)
- [ ] 종업원 급여대장

**E. 사업장 관련**
- [ ] 사업자등록증 사본
- [ ] 임대차계약서
- [ ] 통장 사본 (환급용)

### [2] 프리랜서 — 종합소득세

- [ ] 원천징수영수증 — **거래처에서 발행, 홈택스 자동 조회 가능**
- [ ] 지급명세서 (A) — 홈택스 자동
- [ ] 3.3% 원천징수 내역 — 홈택스 자동
- [ ] 경비 영수증 — **korean-jangbu-for 자동 분류** ✅
- [ ] 연금저축·IRP 납입 확인서
- [ ] 건강보험·국민연금 납부 내역
- [ ] 기부금 영수증

**필요경비율 기준 (추계신고 시)**
- 인적용역(강연·컨설팅·디자인·번역 등): **단순경비율 64.1% / 기준경비율 49.7%** (2025귀속)
- 실제 경비가 단순경비율보다 크면 **장부신고**가 유리 → korean-jangbu-for 장부 자동 생성

### [3] 1인 법인 대표

**법인세 신고 (3월) — 법인이 세무사에게**
- [ ] 재무제표 (BS·PL) — **korean-jangbu-for 생성 초안** ✅ (세무사 검토 필수)
- [ ] 분개장 — **korean-jangbu-for export_djournal** ✅
- [ ] 총계정원장 — 세무사
- [ ] 세무조정계산서 — **세무사 전용**
- [ ] 이익잉여금처분계산서
- [ ] 법인세 과세표준 및 세액신고서

**대표자 종합소득세 (5월) — 본인이 직접**
- [ ] 근로소득 원천징수영수증 (급여 받는 경우)
- [ ] 배당소득 — 법인 배당 있으면
- [ ] 기타소득 — 법인과 별개 프로젝트 수익
- [ ] 건강보험·국민연금 납부
- [ ] 연금저축·IRP
- [ ] 의료비·교육비 공제

### [4] 겸업 (근로소득 + 사업소득)

[1]의 모든 항목 + 아래 추가:
- [ ] 근로소득 원천징수영수증 (회사에서 발급)
- [ ] 연말정산 결과서
- [ ] 합산 신고 대상 여부 확인 (사업소득이 기타소득이 아닌지)

### [5] 성실신고확인대상 개인사업자 (종소세 6월)

[1]의 모든 항목 + **세무사 확인 필수 추가 서류**:

- [ ] **성실신고확인서** — 세무대리인(세무사·회계사)이 발급
  - 매출·비용 전반을 세무사가 실사 후 날인
  - 세무사 수수료: 매출 규모에 따라 50~300만원
- [ ] **수입금액명세서** — 업종별 구분
- [ ] **매입매출장** — 월별 상세
- [ ] **재고자산 명세서** (도소매·제조업)
- [ ] **감가상각자산 대장** (상세)
- [ ] **차입금·예금 상세내역**
- [ ] **임직원 급여대장**
- [ ] **해외거래명세** (있는 경우)

**주의사항**:
- 6/30까지 신고 + 세무사 확인서 동시 제출
- **일반 신고로 제출하면 가산세 5%** (성실신고 대상자 적발 시)
- 세무사 섭외는 **4월 초까지** 권장 — 5월 중순 이후엔 세무사 일정 꽉 찬다
- `jangbu-tax` 생성 PL·BS는 **세무사 검토 전 참고용**, 성실신고확인서 자체를 대체 불가

## korean-jangbu-for 연동

**자동 생성 가능한 항목** (이 스킬 기본 제공):

| 서류 | MCP 도구 | 비고 |
|------|---------|------|
| 간편장부·복식부기 초안 | `export_djournal` | 더존·세무사랑 호환 CSV |
| 손익계산서 초안 | `export_report(pl)` | JSON/CSV/HTML |
| 재무상태표 초안 (간이) | `export_report(bs)` | 정식 BS 아님 주의 |
| 월별 손익 | `export_report(monthly_pl)` | |
| 현금흐름 | `export_report(cash_flow)` | |
| 카드 매입 분류 | `classify_with_rules` | 영수증·카드명세서 포함 |
| 영수증 자동 분류 | `ocr_document(receipt)` | |

## 실행 흐름

### Step 1. 체크리스트 출력

사용자가 주체·연도 선택 후 해당 유형의 전체 체크리스트 표시.

### Step 2. korean-jangbu-for 자동 생성 가능 항목 체크

```python
# 체크리스트에 해당하는 항목 자동 생성
export_djournal(period_start="2025-01-01", period_end="2025-12-31")
export_report(report_type="pl", period_start="2025-01-01", period_end="2025-12-31", fmt="csv")
export_report(report_type="bs", as_of="2025-12-31", fmt="csv")
```

결과 파일 경로를 체크리스트에 매핑:
```
✅ 간편장부: /reports/journal_2025-01-01_2025-12-31.csv
✅ 손익계산서: /reports/pl_2026-04-22.csv
⚠ 재무상태표: /reports/bs_2026-04-22.csv (간이 버전 — 세무사 검토 필수)
```

### Step 3. 수동 준비 항목 안내

### 홈택스 다운로드 경로 (2026년 기준)

각 항목에 ✅ 표시는 **CODEF 자동 수집 지원** (Phase B · jangbu-connect 설정 필요),
⏳ 는 수동만 가능.

**로그인 → 메인 → "조회/발급" 메뉴부터 시작**

#### 매출·매입 자료
```
[ ] ✅ 전자세금계산서 (매출)
    홈택스 → 조회/발급 → 전자(세금)계산서 → 월별 조회·발급
    → 연도 선택 → "매출" 탭 → 엑셀 다운로드
    자동: codef_fetch_hometax(doc_type="tax_invoice_issued")

[ ] ✅ 전자세금계산서 (매입)
    홈택스 → 조회/발급 → 전자(세금)계산서 → 월별 조회·발급
    → "매입" 탭 → 엑셀 다운로드
    자동: codef_fetch_hometax(doc_type="tax_invoice_received")

[ ] ✅ 현금영수증 매출·지출증빙
    홈택스 → 조회/발급 → 현금영수증 → 현금영수증 조회
    → "매출 내역" / "지출증빙 내역" 각각 다운로드

[ ] ✅ 신용카드 매출전표 (가맹점)
    홈택스 → 조회/발급 → 사업용신용카드 → 매출자료 조회
    → 연도·분기 선택 → 엑셀 다운로드

[ ] ⏳ 해외매출 (수출실적명세서)
    관세청 UNIPASS (unipass.customs.go.kr) → "수출신고필증" 조회
```

#### 납세·증명서류 (자동 수집 모두 지원 ✅)
```
[ ] ✅ 소득금액증명
    홈택스 → 민원증명 → 소득금액증명
    자동: codef_fetch_hometax(doc_type="income_proof", year=2025)

[ ] ✅ 납세증명서
    홈택스 → 민원증명 → 납세증명서 (체납 내역 확인)
    자동: codef_fetch_hometax(doc_type="tax_clearance")

[ ] ✅ 사업자등록증명
    홈택스 → 민원증명 → 사업자등록증명
    자동: codef_fetch_hometax(doc_type="biz_reg_proof")

[ ] ✅ 부가가치세 과세표준증명
    홈택스 → 민원증명 → 부가가치세 과세표준증명
    자동: codef_fetch_hometax(doc_type="vat_base_proof")

[ ] ⏳ 표준재무제표증명 (법인 전용)
    홈택스 → 민원증명 → 표준재무제표증명
```

#### 원천징수·인건비
```
[ ] ✅ 원천징수영수증 (근로·사업·기타소득)
    홈택스 → 조회/발급 → 원천징수관련 → 원천징수영수증 조회
    → 종별·연도 선택 → PDF/엑셀 다운로드

[ ] ⏳ 지급명세서 제출현황
    홈택스 → 조회/발급 → 원천징수관련 → 지급명세서 제출현황
    → 귀속연도 선택

[ ] ⏳ 간이지급명세서 제출현황
    홈택스 → 조회/발급 → 원천징수관련 → 간이지급명세서
    (반기별 제출 — 1월·7월)
```

#### 소득공제 자료 (연말정산용)
```
[ ] ✅ 연말정산 간소화 자료

    → 근로자용 / 사업자·기타소득자용 선택
    → 의료비·교육비·기부금·연금저축·주택청약 등 일괄 조회

[ ] ✅ 신용카드 사용액 (공제용)
    홈택스 → 연말정산간소화 → "신용카드 등 사용금액"
```

#### 4대보험·공공
```
[ ] ✅ 국민연금 납부내역
    국민연금공단 (nps.or.kr) → 증명서 발급 → 가입증명서
    자동: codef_fetch_hometax(organization="nps")  # NPS

[ ] ✅ 건강보험 납부내역
    건강보험공단 (nhis.or.kr) → 민원여기요 → 증명서 발급
    자동: codef_fetch_hometax(organization="nhis")

[ ] ⏳ 고용·산재보험
    근로복지공단 (comwel.or.kr) → 증명원 발급
```

#### 금융 자료 (자동 수집 우선 사용)
```
[ ] ✅ 은행 거래내역 (전체 계좌)

[ ] ✅ 카드 이용내역 (매입분)
    자동: codef_fetch_card(issuer="shinhan"|"kb"|..., ...)

[ ] ⏳ 증권 매매내역 (주식·펀드)
    증권사별 HTS/MTS → 매매내역·배당내역 조회

#### 수동 준비 (자동 수집 불가)

### Step 5. 세무사 전달 패키지 생성 (선택)

```
세무사에게 전달할 파일 묶음을 생성합니다.

생성 파일:
